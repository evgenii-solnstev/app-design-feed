# Design Feed — плагин Figma

Отправляет выбранный фрейм в ленту [Design Feed](https://app-design-feed.vercel.app).

**Описание:** Поделись апдейтом с командой. Фрагмент решения появится в ленте.

## Установка и тест (локально)

1. Открой [Figma](https://figma.com) (браузер или приложение) и любой файл.
2. Меню **Plugins** → **Development** → **Import plugin from manifest…**
3. В диалоге выбери папку **figma-plugin** в проекте (или файл `figma-plugin/manifest.json`) и подтверди.
4. Плагин появится в **Plugins** → **Development** → **Design Feed**. Запусти его.
5. **Проверка:** выдели фрейм на холсте → запусти **Design Feed** → нажми **Поделиться в ленте**. Фрейм должен появиться на https://app-design-feed.vercel.app

## Как пользоваться

1. Выбери на холсте фрейм (или любой узел).
2. Запусти плагин **Design Feed**.
3. Нажми **Поделиться в ленте** — фрейм экспортируется в PNG (2×) и отправляется в ленту. Имя автора подставляется из твоего аккаунта Figma (логин/имя). Ссылка на узел в Figma отправляется автоматически.

## Для разработки с локальным API

В `manifest.json` указаны `devAllowedDomains` для `localhost:3000`. Чтобы плагин ходил на локальный сервер, в `ui.html` замени `API_BASE` на `http://localhost:3000`.

## Публикация

При публикации в Community Figma попросит уникальный **id** в манифесте (числовой). Создай плагин через **Create new Plugin** в Figma, скопируй выданный `id` в `manifest.json`.
